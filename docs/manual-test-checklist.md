# 手動テストチェックリスト

## 目的

実装後に、英語しかできない初訪日の観光客が、主要な移動シナリオを迷わず進めるかを確認する。

このチェックリストは、`scenario-test.md` の仕様ベーステストを、実画面のクリック確認へ落としたもの。

## 共通判定

| 判定 | 意味 |
|---|---|
| OK | 迷わず進める |
| WARN | 進めるが、文言やボタン位置に改善余地あり |
| NG | 次の行動が分からない、または戻れない |

## 共通確認項目

- [ ] UIの主表示が英語になっている。
- [ ] 日本語は人に見せる文として表示されている。
- [ ] 日本語文には英語訳が併記されている。
- [ ] 現在の目的地、地域、次に行く場所が画面内で確認できる。
- [ ] 迷った時に何をするかが `If unsure` で分かる。
- [ ] `Do this now` の行動で詰まった時に見せる日本語文が同じ画面にある。
- [ ] 1画面の主ボタンが3つ程度に収まっている。
- [ ] 長い説明が主画面を圧迫していない。
- [ ] `Open Google Maps` の後に `BackFromMapsScreen` へ戻れる。
- [ ] 迷った時に `Show destination in JP`、`Open Google Maps`、`Switch to taxi` のいずれかへ進める。
- [ ] 夜間・本数少ない地域の警告が補足に隠れていない。
- [ ] 「その他の移動手段」は詳細Flowに入らず、Google Maps・係員・公式案内へ渡せる。

## S01: 普通電車で観光地へ行く

- [ ] 目的地確認から開始できる。
- [ ] Final destination入力前に、Google Mapsで目的地を調べる説明がある。
- [ ] `Destination name` だけが必須だと分かる。
- [ ] 都道府県・市区町村などの地域ヒントは推奨の補助項目だと分かる。
- [ ] 目的地種別や次に行く場所にはOptionalラベルがなく、入力を促す表示になっている。
- [ ] Google Mapsで表示された場所が、実際に行きたい目的地か確認できる。
- [ ] 同名の別施設・別地域だった場合に、目的地名または地域を修正できる。
- [ ] 徒歩準備で `Open Google Maps` へ進める。
- [ ] Google Mapsから戻った後、駅に着いた確認へ進める。
- [ ] 普通電車準備で、切符/IC、ホーム、方向、停車駅を確認できる。
- [ ] 急行・快速が止まるか確認する日本語文を表示できる。
- [ ] 最後の徒歩区間確認で、入口・受付・集合場所を確認できる。

## S02: 路線バスで観光地へ行く

- [ ] 路線バス準備が固定順で進む。
- [ ] 現金・小銭準備が先に出る。
- [ ] Google Mapsの降車バス停とルート詳細を確認する導線がある。
- [ ] バス停方向確認で、停留所名だけでなく方向と降車停留所を確認できる。
- [ ] 並んでいる人に見せる日本語文を表示できる。
- [ ] 運転手向けの確認文が短い。
- [ ] ICカードを前提にした選択肢が出ていない。

## S03: バス停名は正しいが方向が違う

- [ ] バス停確認で方向と降りる停留所を一緒に確認できる。
- [ ] 違う場合にGoogle Mapsへ戻れる。
- [ ] タクシー切り替えも選べる。

## S04: バスが満員で乗れない

- [ ] 次の同じ系統・行き先のバスを待つ選択肢がある。
- [ ] 目的地が遠くなければタクシー切り替えを提案できる。
- [ ] 次のバスも混雑する可能性がある理由が短く表示される。

## S05: 夜間・本数が少ない地域

- [ ] 警告が画面上部に出る。
- [ ] 次の便と最終便を確認する導線がある。
- [ ] `Switch to taxi` が見える位置にある。
- [ ] 「1本逃すと数時間後、または今日中に移動できないことがある」が必要時に表示される。

## S05-A: 高速バス・空港バス・夜行バス

- [ ] 路線バスとは別Flowに入る。
- [ ] 高速バス、空港バス、夜行バスを選べる。
- [ ] 予約済みか、乗車券を買うかを選べる。
- [ ] 乗車券、QR、予約画面、予約メール、予約番号を見せる導線がある。
- [ ] 便名、バス会社、乗り場、集合時刻、出発時刻、到着地をチェックリストで確認できる。
- [ ] 荷物を預ける場所とタイミングを確認できる。
- [ ] 乗車直前確認が短い。
- [ ] 変更、キャンセル、返金、次便はアプリで断定せず、カウンター/バス会社へ渡す。

## S06-S07: タクシー

- [ ] 最終目的地まで行くか、途中地点まで行くかを選べる。
- [ ] ホテルスタッフ向けとドライバー向けの表示を分けられる。
- [ ] ドライバー向けには、タクシーで実際に行く場所だけを大きく表示できる。
- [ ] 入口近くで降りたい文を表示できる。

## S08: 普通電車の急行・快速確認

- [ ] ホームと方向を確認できる。
- [ ] 急行・快速が目的駅に止まるか確認する文を表示できる。
- [ ] 有料特急・ライナー・座席指定列車ではないか確認できる。

## S09: Google Mapsから戻った後

- [ ] `I arrived at a station or stop` を選べる。
- [ ] `Check before boarding` を選べる。
- [ ] `Choose next transportation` を選べる。
- [ ] どこへ戻ればよいか迷わない。

## S10: Google Mapsのピンには着いたが入口が分からない

- [ ] `I arrived` の前に最後の徒歩区間確認が出る。
- [ ] 入口、受付、集合場所、ランドマークを確認できる。
- [ ] 人に見せる日本語文を表示できる。
- [ ] 人に見せる日本語文は、原則「〇〇したいです」で目的が短く伝わる。

## S11: 通信不良

- [ ] 保存済みの目的地名を表示できる。
- [ ] 次に向かう場所を表示できる。
- [ ] 人に見せる日本語文を表示できる。
- [ ] Google Mapsが開けなくても人に聞ける。

## S12-S13: 高齢・疲労・荷物が多い・子連れ

- [ ] 明示的にタクシーへ切り替えられる。
- [ ] タクシー目的地を途中地点にできる。
- [ ] 人数・荷物の文を表示できる。
- [ ] 自動最適化がなくても復帰できる。

## S14: 日本語を見せる不安

- [ ] 日本語文の下に英語訳がある。
- [ ] 人に見せるボタンは、`Show destination in JP` など、何を日本語で見せるかが分かる。
- [ ] 表示対象が分かる。
- [ ] 困った時に、この画面を誰に見せればよいか分かる。
- [ ] 困った時に使う日本語文を、別画面に探しに行かなくてよい。

## S15: 間違えた交通手段を選んだ

- [ ] Google Mapsへ戻れる。
- [ ] `Choose next transportation` で選び直せる。
- [ ] 戻る操作でループが壊れない。

## S16: 地方・過疎地のローカルバス/普通電車

- [ ] 交通手段を選んだ直後に警告を出せる。
- [ ] Google Mapsから戻った直後にも次の便・最終便確認へ進める。
- [ ] 不安な場合、乗り場へ向かう前にタクシーへ切り替えられる。

## S17: 路面電車

- [ ] バス種別選択から路面電車を選べる。
- [ ] 路線バス簡易フローへ進む。
- [ ] 系統番号、整理券、降車ボタンの説明が増えすぎていない。
- [ ] 方向、行き先、降りる停留所、支払い方法を優先確認できる。

## S17-A: モノレール

- [ ] 普通電車・地下鉄・モノレールを選べる。
- [ ] 駅、改札、ホーム、方向、降りる駅を確認できる。
- [ ] 空港や施設の場合、ターミナル・入口を確認できる。

## S18-S19: 新幹線・特急

- [ ] 席種選択で自由席、指定席、不明を選べる。
- [ ] 最小画面セット上は同じ `新幹線・特急準備` として表示内容を切り替えられる。
- [ ] 自由席では、乗車券、特急券、自由席車両、乗車位置、停車駅を確認できる。
- [ ] 指定席では、列車名・号数、出発時刻、号車、座席番号を確認できる。
- [ ] 荷物置き場を確認できる。
- [ ] 乗り遅れや変更はアプリで断定せず、窓口・係員へ渡せる。

## S20-S23: その他の移動手段

- [ ] その他を選ぶと、詳細手続きは案内しないことが短く表示される。
- [ ] Google Mapsを開ける。
- [ ] 窓口、係員、公式案内、サービスアプリで確認する導線がある。
- [ ] 確認することが短いチェックリストで表示される。
- [ ] 個別Flowに入りすぎない。

## 完了条件

- [ ] すべてのOK必須シナリオで次の行動が分かる。
- [ ] WARNは実装メモとして残っている。
- [ ] NGがない。
- [ ] 主画面の情報量が多すぎない。
- [ ] 人に見せる文がすぐ出せる。
